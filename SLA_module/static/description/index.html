SLA module

Módulo para actualizar el campo pickup date de acuerdo a los horarios definidos por marketplace.

Descripción

Este modulo crea una vista dentro de la aplicación de ventas para colocar los horarios de pickup date de acuerdo al marketplace que ejecute una orden. De acuerdo a los valores de ese modelo, se actualiza el campo de pickup date del modulo de stock.picking. Existen valores definidos (fijos) dentro del módulo, pero la mayoría (los que peuden variar) se obtienen de la vista mencioanda anteriormente.
